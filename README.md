# 🚀 Активация подписки VLESS на V2rayN и V2rayNG

## 📥 Шаг 1: Скачивание необходимых приложений

### Для Windows (V2rayN)
- Перейдите на официальную страницу релизов проекта [V2rayN](https://github.com/2dust/v2rayn/releases)  
- Выберите последнюю версию файла с расширением .zip или .exe  
- Распакуйте архив в удобную папку  

### Для Android (V2rayNG)
- Перейдите на страницу релизов проекта [V2rayNG](https://github.com/2dust/v2rayNG/releases  )
- Скачайте актуальный файл .apk  
- Установите приложение на ваше устройство  

## ⚙️ Шаг 2: Настройка подключения

### В V2rayN (Windows)
1. Запустите программу от имени администратора  
2. Нажмите правой кнопкой мыши в области уведомлений и выберите "Import from clipboard" или "Import from URL"  
3. Вставьте ссылку на вашу подписку VLESS  
4. Выберите сервер из списка и нажмите "Set as default proxy server"  

### В V2rayNG (Android)
1. Откройте приложение  
2. Нажмите на значок "+" в правом нижнем углу  
3. Выберите "Import URL" или "Import clipboard"  
4. Вставьте ссылку на вашу подписку VLESS  
5. Подтвердите импорт и активируйте подключение 

## 💡 Прочитайте обязательно!
- Регулярно обновляйте приложения до последних версий
- Обязательно используйте раздельное проксирование для корректной работы приложения  
- При проблемах с подключением попробуйте переключить протокол
